El Ejemplo

Este aplicativo tiene como finalidad demostrar las operaciones mas comunes y repetitivas que se me ocurrieron implementadas en apache wicket , que es el framework que utilizaremos para la gran mayoria de los proyectos.
Por ahora este ejemplo cuenta con :
- Altas : crear un ticket
- Bajas : borrar un ticket
- Modificaciones : modificar un ticket
- Listado : listar todos los tickets existentes
- Busquedas : listar solo los tickets que contengan el string ingresado en el campo de search

A medida que se vayan agregando funcionalidades al ejemplo se iran agregando en este listado para un mejor seguimiento.
